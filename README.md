# WebStack 网址导航（EdgeOne Pages + KV 版）

在 WebStack-Hugo 主题基础上改造的书签导航站：书签数据、天气配置等持久化数据全部存储在 **EdgeOne KV 空间**，通过 EdgeOne Pages Functions 提供 API，支持在线添加、右键/长按编辑、删除书签，图标自动通过 API 获取。

## 功能

- 📖 书签分类与卡片全部从 KV 动态读取（首次访问自动写入内置种子数据，之后以 KV 为准）
- ➕ 右下角浮动按钮添加书签（分类下拉、备用链接、标题识别、私有开关、自定义图标、连续添加）
- 🖱️ 桌面**右键**、移动端**长按**书签卡片 → 访问 / 复制 / 编辑 / 删除
- 🖼️ 书签图标留空自动走 `/api/icon` 接口抓取 favicon（KV 缓存 7 天，抓取失败回退字母图标）；也可上传自定义图标
- 🌦️ 和风天气：Key 存 KV 不暴露给前端，Function 代理调用并缓存 30 分钟；页头齿轮随时修改
- 🔐 写操作需管理密码（`X-API-Key` 认证），私有书签仅管理员可见；纯访客只读
- 🔎 标题「识别」按钮自动抓取网页标题（支持 GBK 编码的中文站点）

## 部署到 EdgeOne Pages

### 方式一：Git 导入（推荐）

1. 把整个目录推送到你的 GitHub/GitLab 仓库。
2. EdgeOne Pages 控制台「创建项目」→「导入 Git 仓库」。
3. 框架预设选「**静态网站**」，**构建命令留空**，输出目录填 `.`。
4. 部署后**必须完成两项配置**（见下）。

### 方式二：直接上传

控制台「创建项目」→「上传文件」，上传本目录全部内容。

### 必做配置（项目设置）

| 配置项 | 位置 | 值 |
| --- | --- | --- |
| KV 绑定 | 项目设置 → KV 存储 → 绑定变量 | 变量名必须为 **`BOOKMARK_KV`**，选择你的 KV 命名空间 |
| 管理账号 | 项目设置 → 环境变量 | `AUTH_USERNAME` = 你的登录账号（**可选**，不设置则登录只校验密码） |
| 管理密码 | 项目设置 → 环境变量 | `AUTH_PASSWORD` = 你的登录密码（**必须**） |

> 未绑定 KV 时页面会提示无法加载；未配置 AUTH_PASSWORD 时所有写操作被拒绝（只读模式）。
> 修改任一环境变量后，已签发的登录令牌自动失效（需重新登录）。

## 安全验证

- **登录场景**：添加、编辑、删除书签，查看私有书签，修改站点设置，均需先通过账号密码登录。
- **登录方式**：点击右下角 🔒 按钮（或进行受限操作时自动弹出），输入 `AUTH_USERNAME` / `AUTH_PASSWORD`。
- **令牌机制**：登录成功后服务端签发 HMAC-SHA256 签名令牌（**7 天有效**），保存在浏览器 localStorage，后续请求自动携带；不在请求中直接传输密码。
- **退出登录**：点击右下角 ✓ 按钮，退出后私有书签立即隐藏。
- **私有书签**：未登录访客的数据请求在服务端即被过滤，私有书签不会下发到浏览器。

## 使用说明

- **添加书签**：点右下角 ➕ → 首次会要求输入管理密码（存浏览器会话）→ 填写表单。「所属分类」可选现有分类或「新建分类…」（格式：`顶级分类` 或 `顶级分类/二级分类`）。
- **编辑/删除**：右键（手机长按）书签卡片。
- **修改天气/站点标题**：页头天气旁的 ⚙ 齿轮。和风 2024 年后新建的 API 项目需在「API 域名」填控制台里的专属域名（如 `xxx.re.qweatherapi.com`），旧项目留空即可。
- **私有书签**：开关打开后，未验证密码的访客完全看不到该书签。

## API 一览（供参考）

| 方法 | 路径 | 说明 | 认证 |
| --- | --- | --- | --- |
| GET | `/api/data` | 分类+书签+公开设置 | 可选（带 key 含私有） |
| POST | `/api/login` | 校验密码 | - |
| POST | `/api/bookmarks` | 添加书签 | ✅ |
| PUT | `/api/bookmarks/:id` | 更新/移动书签 | ✅ |
| DELETE | `/api/bookmarks/:id` | 删除书签 | ✅ |
| GET/PUT | `/api/settings` | 读写设置 | ✅ |
| GET | `/api/weather?city=` | 和风天气代理（缓存 30 分钟） | - |
| GET | `/api/icon?url=` | favicon 抓取（KV 缓存 7 天） | - |
| GET | `/api/title?url=` | 网页标题识别 | - |

## 数据说明

- KV 键 `nav:data`：分类树 JSON；`nav:settings`：天气 Key/城市/站点标题；`icon:*`：图标缓存；`nav:weather-cache`：天气缓存。
- 更换站点数据不需要重新部署，改完即时生效。
